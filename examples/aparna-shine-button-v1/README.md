# Shine Button Utility

A reusable CSS utility that creates a smooth shine animation across a button on hover.

## Features

- Lightweight
- Pure HTML & CSS
- Reusable utility
- Smooth shine animation
- Perfect for CTA buttons

## Usage

```html
<button class="shine-button">
  Get Started
</button>
```
